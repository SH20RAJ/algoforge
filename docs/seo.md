# SEO Strategy

## Technical

- Unique `title` / `description` / H1 per page via `buildMetadata`
- Canonical URLs, Open Graph, Twitter cards
- JSON-LD: Organization, WebSite+SearchAction, SoftwareApplication, Article, FAQ, HowTo, BreadcrumbList
- `sitemap.ts`, `robots.ts`, RSS at `/feed.xml`, HTML sitemap

## Programmatic pages

- Problems (+ language variants)
- Topics, patterns, companies, difficulty, languages
- Collections (Blind 75, Grind 169, NeetCode alternatives)
- Roadmaps, blog, explore landings (company×topic, difficulty×pattern)

## Quality bars

- Landings require ≥3 problems
- Original intro + FAQs + internal links
- No doorway-only pages
